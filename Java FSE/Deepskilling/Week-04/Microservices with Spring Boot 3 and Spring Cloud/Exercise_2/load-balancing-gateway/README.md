# Exercise 2: Load Balancing in an API Gateway

## Objective
Implement load balancing in an API Gateway using Spring Boot 3 and Spring Cloud.

## Technologies Used
- Java 17
- Spring Boot 3
- Spring Cloud Gateway
- Spring Cloud LoadBalancer
- Maven

## Features
- Configured Spring Cloud Gateway
- Implemented custom load balancer configuration
- Configured load balancing properties in `application.properties`

## Build
```bash
mvn clean install
```

## Run
```bash
mvn spring-boot:run
```

## Result
The API Gateway was successfully configured with Spring Cloud LoadBalancer, and the application started successfully.
