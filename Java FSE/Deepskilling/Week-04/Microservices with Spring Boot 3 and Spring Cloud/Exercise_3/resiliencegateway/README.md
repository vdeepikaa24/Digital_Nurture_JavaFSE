# Exercise 3: Resilience Patterns in an API Gateway

## Objective
Implement resilience patterns in an API Gateway using Spring Boot 3, Spring Cloud Gateway, and Resilience4j.

## Technologies Used
- Java 17
- Spring Boot 3
- Spring Cloud Gateway
- Resilience4j
- Maven

## Features
- Configured Resilience4j Circuit Breaker
- Added custom resilience configuration
- Configured circuit breaker properties in `application.properties`

## Build
```bash
mvn clean install
```

## Run
```bash
mvn spring-boot:run
```

## Result
The API Gateway was successfully configured with Resilience4j resilience patterns and the application started successfully.
