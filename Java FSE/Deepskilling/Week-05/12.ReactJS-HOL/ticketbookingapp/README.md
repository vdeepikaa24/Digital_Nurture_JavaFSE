# Ticket Booking App (React)

## Overview
This is a simple React app created with **Create React App**. It demonstrates **conditional rendering** based on a login state:

- If the user is logged out: shows **"Please sign up."** and a **Login** button.
- If the user is logged in: shows **"Welcome back"** and a **Logout** button.

## Project Setup

### 1) Install dependencies
```bash
npm install
```

### 2) Start the development server
```bash
npm start
```
Open:
- http://localhost:3000

## Tests
```bash
npm test
```

## Build for Production
```bash
npm run build
```

## Key File
- `src/App.js` — contains the login/logout state and conditional UI.
