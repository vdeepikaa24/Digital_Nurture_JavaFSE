# CohortsTracker

This project is a React-based application designed to track and display cohort details. It serves as a lab exercise focusing on component architecture, API consumption, and unit testing using Enzyme and Jest.

## Features
- Displays a list of cohorts with details including:
  - Cohort ID and Program Name
  - Start Date
  - Current Status
  - Coach and Trainer information

## Tech Stack
- **Frontend**: React.js
- **Testing Framework**: Jest and Enzyme
- **Development Tools**: Node.js, npm
