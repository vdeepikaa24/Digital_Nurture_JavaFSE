# Student Score Calculator (React)

This is a React application built with **Create React App**. It demonstrates the use of **Functional Components** and external CSS styling to display student academic details.

- **CalculateScore** (`src/Components/CalculateScore.js`) - A functional component that accepts student data via props and calculates the average score.

---

## Project Structure

- `src/App.js` - Root component that passes student data to the `CalculateScore` component.
- `src/Components/CalculateScore.js` - The main functional component for data display and calculation.
- `src/Stylesheets/mystyle.css` - External CSS for styling components.

---

## Available Scripts

In the project directory, you can run:

Runs the app in development mode.

- Open: http://localhost:3000

### `npm test`

Launches the test runner in watch mode.

### `npm run build`

Builds the app for production to the `build` folder.
