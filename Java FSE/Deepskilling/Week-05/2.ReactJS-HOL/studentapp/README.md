# Student Management Portal (React)

This is a small React application built with **Create React App**. It renders three pages/components on the screen:

- **Home** (`src/Components/Home.js`)
- **About** (`src/Components/About.js`)
- **Contact** (`src/Components/Contact.js`)

`src/App.js` imports these components and displays them.

---

## Project Structure

- `src/App.js` - Root component that renders `Home`, `About`, and `Contact`.
- `src/Components/Home.js` - Home page content.
- `src/Components/About.js` - About page content.
- `src/Components/Contact.js` - Contact page content.

---

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode.

- Open: http://localhost:3000

### `npm test`

Launches the test runner in watch mode.

### `npm run build`

Builds the app for production to the `build` folder.

---

## Notes

This project currently uses simple class-based components for the Home/About/Contact sections.
