# Counter App (React)

A simple **people counter** built with **React**.

## How it works
- **Login** button: increments **People Entered**
- **Exit** button: increments **People Left**

## Run the app
```bash
cd counterapp
npm install
npm start
```
Open http://localhost:3000

## Project structure
- `src/App.js` — renders the counter component
- `src/CounterPeople.js` — counter state + UI (Login/Exit)
