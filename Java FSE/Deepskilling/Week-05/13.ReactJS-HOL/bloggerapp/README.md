# Blogger App (React)

This project is a simple Create React App (CRA) that displays **Course Details**, **Book Details**, and **Blog Details** in a 3-column layout.

## Features
- 3-column UI layout (each column contains related details)
- Styling via `src/App.css`

## Project Setup

### 1) Install dependencies
```bash
npm install
```

### 2) Run the development server
```bash
npm start
```
Then open:
- http://localhost:3000

## Available Scripts
- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects CRA configuration (one-way)

## Project Structure (high level)
- `src/App.js` - Main UI for the 3-column layout
- `src/App.css` - Styles for the layout
- `src/Data.js` - Data module used by the app (if integrated)

## Notes
This app was bootstrapped with **Create React App**.
