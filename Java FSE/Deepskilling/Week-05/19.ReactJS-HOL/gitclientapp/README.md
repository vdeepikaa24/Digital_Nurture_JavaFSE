# GitClientApp

A React application demonstrating API integration with the GitHub API using `axios` and unit testing with Jest mocking.

## Project Setup & Execution
1.  **Install dependencies**: `npm install axios`
2.  **Start the app**: `npm start`
3.  **Run tests**: `npm test`

## Key Functionalities
- **GitClient.js**: Encapsulates GitHub API calls using `axios`.
- **App.js**: Retrieves and renders repository lists.
- **GitClient.test.js**: Validates service logic by mocking `axios` to ensure isolated, deterministic tests.
